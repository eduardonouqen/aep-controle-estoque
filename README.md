# Controle de Estoque de Alimentos para ONGs

Segue a versão dinal do código utilizado para a AEP 2024.1.2 da UniCesumar

Para rodar o código, insira no console:

    1- npm install
    2- node index.js
    3- npm start
    4- Press w │ open web
